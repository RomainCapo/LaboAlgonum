Contextualisation

Dans le cadre du cours d'algorithme numérique, il nous a été demandé de déterminer les solutions d'un système d'équations linéaires. La matrice se trouve stocké dans un fichier JSON qu'il faut lire et converir en matrice à l'aide d'un tableau 2 dimension. Nous devons par la suite triangularisé cette matrice afin de pouvoir trouver chaques solutions.

Méthodologie

On nous a donc fourni avec divers système d'équations linéaires sous forme de matrice. Nous avons utilisé la méthode de Gauss afin de trouver les solutions de ces matrices fournies. Nous avons d'abord créé une fonction afin de transformer les matrices données en matrices triangulaires. Depuis ces matrices, nous pouvions trouver les solutions en resolvant les équations.

Pour résoudre ce problème, nous avons utilisé 2 classes :

  • Matrix :

    Cette classe s'occupe de gérer le stockage de la matrice et contient les principales méthodes qui permet d'effectuer les traitements sur la matrice.

    • constructor(data) :

      Prends comme paramètre le flux JSON. Par la suite on génére la matrice à partir du fichier JSON. On place les valeurs de la matrice ainsi que le vecteur B dans un tableau 2 dimsensions contenu dans this.matrix.

    • argmax(indexColumn, indexLineStart) :

      Permet de trouver la valeur maximal dans une collone de la matrice, renvoie ensuite l'index, le 1er arg est l'index de la colonne et le second l'index de la ligne ou l'on commence a prendre en compte le maximum.

    • swapRows(i,j) :

      Echange 2 ligne de la matrice selon les indice spécifiés en parametres.

  • MatrixSolver :

    Cette calsse permet de résoudre le système d'equation a partir d'une matrice

    • constructor(matrix) :

      On passe en argument la matrice. On s'occuper ensuite d'appeler les methodes de triangularisation et de résolution. Pour finir, on traite le cas ou la matrice n'a aucune solution et on calcul le temps qu'a pris l'algorithme pour trouver les solutions.

    • _gaussMatrixTransform(matrix) :

      Permet de triangulariser la matrice. Nous avons utilisé le pseudo code de la page wikipédia sur l'elimination de gauss afin de résoudre ce problème. On passe en parametre la matrice. Nous utilisons les différentes fonctions de la classe matrice pour effectuer les transformations sur celle-ci.

    • _findSolution(matrix) :

      Permet de trouver les solutions du système d'équation a partir d'une matrice triangularisé. Nous utilisons la méthode vu en cours pour trouver les solutions.

    • _displaySolutions(equationSolutionsArray) :

      Permet d'afficher les solutions du système d'équation. On passe en parametre les solutions du système.

Conclusion

La méthode utilisée est très rapide au niveau du calcul, même pour des grandes matrices. De plus, le fait d'obtenir une matrice triangulaire peut s'avérer très pratique comme quand on doit calculer le détérminant. Nous avons effectué les tests sur les matrices fourni et nous arrivons a des résultats conculants.
