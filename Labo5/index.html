Contextualisation

Le but de ce laboratoire, est d'approximer le nombre PI avec la forumle suivante :

.

L'approximation de PI doit se faire avec 17 décimales. Il est également demandé de calculer le temps de l'opération. Pour calculer cette approximation, nous devons utiliser la formule de simpson qui permet de calculer une intégrale.

Methodologie

Nous avson defini 2 fonctions :

  • simpson(f, a, b, n) : cette meéthode permet d'effectuer l'approximation d'une intégrale avec la méthode de simpson. Le premier parametre est la fonction à intégrer. Le parametre a est la borne inférieur. Le parametre b est la borne supérieur. Le parametre n est le nombre d'itération. La fonction retourne un tableau contenant le resultat ainsi que le temps de calcul effectué.
  • calulatePi() : C'est la fonction qui permet d'appeler la méthode de simpson avec les différentes parametres. Dans notre cas, la fonction definit plus haut. Les bornes de 0 à 1 et 500 itérations pour l'approximation.
    • On termine en affichant le resultat de l'approximation et le temps de calcul.

Conclusion

On peut remarquer, que selon nos résultat l'approximation de PI est précise jusqu'a la 15 décimales. L'implémentation que nous avons réalisé nous semble donc correcte. De plus elle permet d'etre utilisé avec n'importe quelle fonction et n'importe quelle borne.
